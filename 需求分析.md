# 需求分析

## 界面的模块

- 顶部栏
- 左侧 - 组件库
- 左侧 - 图层
- 中间 - 画布
- 右侧 - 属性
- 右侧 - 页面设置

## 功能列表

顶部栏
- 返回
- 显示标题，修改标题
- 工具栏
    - 删除
    - 隐藏
    - 锁定
    - 复制，粘贴
    - 上移，下移
    - 撤销，重做
- 保存，自动保存，ctrl+s快捷键
- 发布

左侧 组件库
- 显示组件列表
    - （各个组件，看系统）
- 点击添加组件到画布

左侧 图层
- 显示图层列表
- 拖拽排序
- 单击，选中
- 双击，修改标题
- 隐藏
- 锁定

中间 画布
- 展示组件列表
- Y滚动条
- 拖拽排序
- 单击，选中
- 快捷键
    - delete backspace
    - up 
    - down
    - ctrl+c,v
    - ctrl+z,ctrl+shift+z
    - ctrl+s,保存

右侧 属性
- 修改属性

右侧 页面设置
- 标题，描述
- JS，CSS代码
- tab自动切换
